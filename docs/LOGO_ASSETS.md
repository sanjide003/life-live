# Life OS Logo Asset Specification

A custom logo package is required before final launcher icon and splash screen setup.

## Required files

Place these files in the project `assets/` folder:

| File | Purpose | Required size / format |
| --- | --- | --- |
| `app_logo.svg` | Master vector source | SVG |
| `app_logo_1024.png` | Master PNG | 1024 x 1024 px |
| `app_logo_512.png` | Large PNG | 512 x 512 px |
| `app_logo_192.png` | Android/web icon | 192 x 192 px |
| `app_logo_180.png` | iOS icon size | 180 x 180 px |
| `app_logo_96.png` | Android icon size | 96 x 96 px |
| `app_logo_72.png` | Android icon size | 72 x 72 px |
| `app_logo_48.png` | Android icon size | 48 x 48 px |
| `favicon.ico` | Web favicon | ICO |
| `favicon-32.png` | Web favicon | 32 x 32 px |
| `favicon-16.png` | Web favicon | 16 x 16 px |
| `splash_logo.png` | Splash logo | 1024 x 1024 px, transparent background |
| `adaptive_foreground.png` | Android adaptive icon foreground | 1024 x 1024 px, transparent background |
| `adaptive_background.png` | Android adaptive icon background | 1024 x 1024 px, solid background |

## Design rules

- Modern and minimal.
- Rounded shape.
- Flat design.
- High contrast.
- Recognizable at small sizes.
- Works in both light mode and dark mode.
- No text inside the icon.
