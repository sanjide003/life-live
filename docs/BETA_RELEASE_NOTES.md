# Livelife Beta Release Notes

## Highlights

- Offline-first planning, habits, finance, health, prayer, notes, reviews, and reports.
- Optional Google sign-in and Firebase backup boundaries.
- SQLite local storage foundation.
- Prayer calculation settings and prayer progress tracking.
- Notification, Health Connect, backup, import/export, and onboarding foundations.

## Known production blockers

- Flutter/Dart CI must run successfully in a full Flutter environment.
- Store signing secrets must be configured outside git.
- Firebase emulator security tests should be run before public beta.
- Generated raster logo assets should be produced locally from `assets/app_logo.svg` before packaging.
